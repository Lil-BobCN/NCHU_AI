# Sandbox

Allowed:
- Web research.
- Browser observation of reference sites.
- Write research artifacts under `.omx/specs/autoresearch-homepage-reference-motion-design/`.
- Write planning/log debt artifacts under `.omx/logs/`.
- Update OMX autoresearch state.

Not allowed:
- Editing frontend source code.
- Installing skills or npm packages.
- Downloading production visual assets.
- Implementing the homepage redesign.
