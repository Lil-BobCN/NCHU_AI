# Autoresearch Mission: Frontend Skills Selection

Date: 2026-05-19
Status: completed

## Mission

Find and install frontend-related Codex/agent skills from GitHub that best fit the AI Counselor Demo frontend workflow.

## Selection Criteria

- Must help the current React + TypeScript + Vite + Ant Design / shadcn-ui frontend work.
- Prefer mature or authoritative sources.
- Prefer skills that improve visual quality, component discovery, component review, screenshot/browser QA, or design-system thinking.
- Avoid installing broad, unrelated, or unverified skill packs.
- Avoid Figma-specific skills unless the current workflow needs Figma. The current workflow is website screenshot/reference driven, not Figma-node driven.

## Expected Output

- Installed skills list.
- Why each skill was selected.
- What frontend development phase each skill supports.
- Any limitations or restart requirements.
